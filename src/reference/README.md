# Reference Implementation

This directory contains a copy of the payload-page-tree plugin source code for reference.
It is excluded from TypeScript compilation and the build process.

**DO NOT** modify these files - they are for reference only.

See the patterns in:
- `components/PageTreeView.tsx` - RSC admin view pattern
- `components/PageTreeClient.tsx` - Client component pattern
- `endpoints/treeOperations.ts` - Endpoint handler pattern
- `index.ts` - Plugin configuration pattern
